# DAMPBootstrapTailwind - Project

Repositori ini berisi dua versi tampilan antarmuka aplikasi To Do List yang dikembangkan menggunakan dua framework berbeda: Bootstrap dan Tailwind CSS.

## Struktur Folder
├── bootstrap/
└── tailwind/

### 1. `bootstrap/`
Folder ini berisi implementasi tampilan To Do List menggunakan Bootstrap.  
Disusun oleh: Alexandra Pritha Sari

### 2. `tailwind/`
Folder ini berisi implementasi tampilan yang sama namun menggunakan Tailwind CSS.

## Fitur Tampilan

Kedua versi UI (Bootstrap dan Tailwind) memiliki fitur-fitur dasar yang sama:

- Tampilan daftar tugas awal, lengkap dengan label klasifikasi tugas (seperti penting, mendesak, dan lainnya)
- Checkbox pada tiap tugas, seperti aplikasi To Do List pada umumnya
- Icon silang (cross) untuk menghapus item dari daftar
- Tombol nama penyusun, yang menampilkan dialog informasi penyusun
- Tombol Add yang mengarahkan ke tampilan untuk menambahkan tugas baru

## Cara Menjalankan

1. Klik tombol `Code` (warna hijau) di halaman repository, lalu pilih `Download ZIP`
2. Ekstrak ZIP yang telah diunduh
3. Masuk ke folder `bootstrap/` atau `tailwind/`
4. Klik dua kali file `ToDoApp.html` untuk membuka tampilan di browser
